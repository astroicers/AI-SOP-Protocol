
## TD-004 — Draft 偵測 regex 過寬，可能誤報 [IMPORTANT]

**位置：** `.asp/hooks/session-audit.sh` 第 152 行  
**問題：** `` `Draft` `` backtick 模式會匹配 ADR 內文說明段落（如「`Draft`（初稿）→ `FIRM` → `Accepted`」），不只匹配表格狀態欄  
**修法：** 改為精確匹配表格欄位：
```bash
grep -qiE "\|\s*\`Draft\`\s*\|" "$adr_file"
```
**風險：** 若未來 Accepted ADR 內含流程說明樣板文字，可能被誤判為 Draft 並觸發 commit deny
